== Description ==

This Widget display Instagram thumnail by user id or tag(hash tag). 

== Installation ==

1. Upload `minstagram` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= What is CLIENT ID? =

You must register your wordpress to Instagram Manage Clients page.
CLIENT ID is published at Instagram Manage Clients page.
